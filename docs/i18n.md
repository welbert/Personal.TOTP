# Internationalization (i18n)

## Libraries

| Package | Role |
|---------|------|
| `i18next` | Core i18n engine |
| `react-i18next` | React bindings (`useTranslation`, `Trans`) |

## File structure

```
src/i18n/
├── index.ts       # Initialization — loads resources, detects language
├── pt-BR.json     # Portuguese (Brazil) translations
└── en-US.json     # English (US) translations
```

## Adding a new language

1. Create `src/i18n/<locale>.json` following the same key structure as `en-US.json`
2. In `src/i18n/index.ts`, import the file and add it to the `resources` object:
   ```ts
   import es from "./es.json";
   // ...
   resources: {
     "pt-BR": { translation: ptBR },
     "en-US": { translation: enUS },
     "es":    { translation: es },
   }
   ```
3. Add the locale to the `LANGUAGES` array in `src/components/SettingsModal.tsx`
4. Add its display name under `settings.languages` in every translation file

## Translation key structure

```
setup.*         SetupScreen — vault creation form
unlock.*        UnlockScreen — unlock form
main.*          TotpList — header, search, empty states
entry.*         TotpEntry — favorite/delete tooltips, confirm dialog
addEntry.*      AddEntryModal — add account form
settings.*      SettingsModal — title, language picker
```

## Language persistence

- On init: reads `localStorage.getItem("lang")` → falls back to browser locale (`navigator.language`) → falls back to `en-US`
- On change (via `SettingsModal`): calls `i18n.changeLanguage(code)` + writes to `localStorage`
- Change is immediate — no page reload required

## Using translations in components

```tsx
import { useTranslation } from "react-i18next";

export default function MyComponent() {
  const { t } = useTranslation();
  return <p>{t("main.search")}</p>;
}
```

For interpolated values:
```tsx
t("main.noResults", { query: search })
// en-US: 'No results for "{{query}}".'
```

For inline HTML/JSX (e.g. `<code>` tags), use `<Trans>`:
```tsx
import { Trans } from "react-i18next";

<Trans i18nKey="addEntry.otpauthHint">
  Paste an <code className="text-emerald-400">otpauth://</code> URL ...
</Trans>
```
The `<1>` tag in the JSON maps to the first child element of `<Trans>`.
